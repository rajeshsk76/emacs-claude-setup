# Claude Code Project Context

## Project Overview
This is an Emacs + Claude Code development environment running on GitHub Codespaces.

## Environment
- **Editor**: Emacs (terminal mode)
- **Platform**: GitHub Codespaces (Ubuntu)
- **Claude Integration**: claude-code.el package
- **Shell**: bash

## Coding Standards
- Prefer clean, readable code over clever one-liners
- Add comments to explain non-obvious logic
- Keep functions small and focused

## Emacs Preferences
- Use spaces not tabs
- 2 space indentation for most languages
- 4 space indentation for Python

## What Claude Should Know
- I am learning Emacs as a beginner
- Explain any Emacs-specific concepts clearly
- Prefer simple solutions over complex ones
- When editing config files, always explain what the change does
